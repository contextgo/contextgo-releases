## 什么是定时任务

定时任务用于按固定周期执行一条已保存的工作流。适合放入周期同步、定期审阅、报表生成或重复性的发布准备任务。

每个任务都应明确计划时间、所需运行时、输入来源和预期输出。

## 主机要求

定时任务运行在桌面主机上。启用前请先确认：

- 到点时主机保持开机。
- 所需运行时已经安装并完成认证。
- 任务使用的文件、连接器和凭证在主机上可用。

如果主机离线，任务无法按计划执行。

## 失败处理

要提前说明任务漏跑或执行失败后的行为。至少应让用户知道，下次是重试、跳过，还是仅等待下一次计划时间。

## 推荐用法

定时任务适合输入稳定、步骤固定的工作。对于频繁依赖人工确认的流程，应把人工审批步骤和定时执行部分拆开配置。
